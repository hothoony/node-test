## Node
### Node 설치
```shell
$ brew install node
```

### Node 설치 확인
```shell
$ node -v
v17.6.0
```

## npm

### npm 설치 확인
```shell
$ npm -v
8.5.1
```

### package.json 생성
```shell
## package.json 이 생성된다
$ npm init -y
```

### Node dependencies 설치
```shell
## npm install 실행하면 node_modules 폴더가 다시 생성된다
$ npm install
```
