# node backend
## backend library
- `express` web server
- `morgan` log
- `winston` log
- `ejs` view template
- `nodemon` server auto restart

## npm install
- express
```javascript
npm install express --save
```
- nodemon
```javascript
// nodemon 전역으로 설치
sudo npm install nodemon -g

// nodemon 실행
// 소스 파일이 수정되는 경우 서버를 자동으로 restart 해준다
nodemon ./bin/www.js
```
